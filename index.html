STYLE GUIDE

This page contains specific style guide information for the Barter Bin Web Application.

Logo

The Barter Bin logo should be prominently displayed on all pages of the website. The logo should be placed at the top of the page and should be easily identifiable. The logo should not be stretched, distorted, or altered in any way.


FONT

The application will utlize the listed font families. The main font color for paragraph text will be dark gray. Variations of green, orange, and yellow as displayed in the color theme chart, will be used for header text and other important text being used to draw the users attention.

  • Arial, Helvetica, sans-serif
  • Impact, Haettenschweiler, 'Arial Narrow Bold', sans-serif
